# Specification

## Summary
**Goal:** Add messaging between matched users and photo posting with feed functionality.

**Planned changes:**
- Add backend data models and functions for sending messages between matched users
- Add backend data models and functions for creating and retrieving photo posts
- Build a messaging interface with conversation list and chat view
- Create a photo feed page displaying posts from matched users
- Add a photo posting component with image upload and caption input
- Add navigation links for messages and photo feed in the header

**User-visible outcome:** Users can send messages to their matches through a chat interface, post photos with captions, and view a feed of photos from their matched connections.
